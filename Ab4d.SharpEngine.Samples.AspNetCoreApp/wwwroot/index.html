Ab4d.SharpEngine browser demo without a Blazor

This sample is showing how to show 3D graphics by using Ab4d.SharpEngine library in a simple HTML and JavaScript based web page.


Available actions: (call methods in SharpEngineTest .Net project):
Toggle camera rotation
Change material to Blue
Change material to a random color
Add 100 objects
Add 1000 objects

See also Browser's console to see log messages from JavaScript and .Net.
